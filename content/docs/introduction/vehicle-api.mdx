### API Overview

The Vehicle API dynamically resolves high-quality images for any vehicle type.
Simply pass the vehicle name as a parameter, and the API will return a relevant,
high-resolution stock image.

### Endpoint

| Property | Value |
| --- | --- |
| Base URL | `https://img-server-theta.vercel.app/api/vehicle/:name` |
| Method | `GET` |
