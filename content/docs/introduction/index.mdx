PixStock provides developer-friendly image APIs for documentation, GitHub
READMEs, profile pages, and product surfaces.

## Available APIs

- GitHub Stats API for dynamic profile cards.
- Pin API for repository showcase cards.
- Tech Stack Icons API for branded SVG icon grids.
- Animal API for local-first and Pexels-backed image resolution.

## Adding Pages

Create a new `.mdx` file inside `content/docs`. Fumadocs discovers content from
the source registry and builds the sidebar from the file tree and metadata.
