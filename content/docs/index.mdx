PixStock is asset infrastructure for product teams that need predictable media
storage, fast delivery, and developer-friendly automation.

## Start here

Install the core package and initialize a workspace:

```bash
npm install pixstock-core
npx pixstock init --workspace assets
```

## Why PixStock

- Store and organize production assets with stable identifiers.
- Deliver optimized media through a developer-first API.
- Automate asset ingestion and lifecycle tasks from your application.
